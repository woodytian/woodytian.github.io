WordPress转GitHub+Hexo

2017-01-11

  2. 搭建GitHub+Hexo环境
  3. 同步

    建议Git部分就用cmder，hexo命令再用GitBash

GitHub+Hexo

创建本地环境

  1. 安装Node.js
  2. 安装Git（亲测 cmder 不能用，上传的时候会报错，只能用 git bash）
  3. 在电脑上建一个文件夹用作Blog文件夹，名字随意
  4. 安装Hexonpm install -g hexo
  5. 安装好之后初始化hexo init
  6. 生成静态页面hexo generate
  7. 启动本地服务hexo server
  8. 访问本地页面localhost:4000

配置GitHub

  1. 在GitHub上建立一个Blog用的responsity，和Blog地址同名(woodytian.github.io)

  2. 修改_config.yml文件，注意冒号后边有空格

  3. 保存部署npm install hexo-deployer-git --save

  4. 部署hexo deploy
  5. 访问网页

基础使用说明

每次部署的步骤，可按以下三步来进行

  • hexo clean
  • hexo generate
  • hexo deploy

一些常用命令

  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面
  • hexo generate #生成静态页面至public目录
  • hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
  • hexo deploy #将.deploy目录部署到GitHub
  • hexo help #查看帮助
  • hexo version #查看Hexo的版本
